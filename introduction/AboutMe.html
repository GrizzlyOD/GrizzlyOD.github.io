# 公司介绍(Company introduction)

中山市永发服装道具展示用品有限公司，是一家从事服装模特道具生产有近20年经验的老厂家。 本厂从1996年建厂以来，一直从事服装展示道具的生产。由于我们一直坚持“质量第一，用户至上”的原则，严把质量关，得到广大消费者的肯定和支持，我们厂也由原来十多人的小作坊，扩大到有100多专业生产人员的正规厂家，并在香港成立了分公司：永发模特（香港）有限公司，专门从事服装展示道具的销售，并希望通过这个国际大平台能更早更好地发现时尚化、个性化的产品。 我厂在2006年就开始了机械化生产，PU产品的灌注等已由电脑机器操控代替人工制作，使产品的质量得到更大的保证。由于我们的产品品质卓越、加上价格合理，不但得到各地代销商的肯定，也得到了大服装厂家的支持。产品不但在大陆畅销，而且还远销港澳台、欧美国家和中东地区，在国内外都拥有良好的知名度和声誉。 我们将时刻洞察潮流演变的先机，先人一步推出款式多样的具有个性化、时尚化的产品，希望我们的服装展示模特道具能为你的时装增添更多姿彩！

# 产品介绍

本厂专业生产以下产品： 插针包布半身模特：包括小童半身、男装半身、女装半身等，主要用于上衣、裙子、婚纱等的展示，也可在下面配上裤子、鞋等作全套服装的展示。用环保PU料，经过特殊加工一次成型，可插针，包布颜色可选择，头可根据需要更换。可配海绵软手、ABS胶手、原木手。手可随意摆出各种姿势。可插于支架上，也可摆放于台上。 按照类别可以分为：

# 玻璃钢半身模特

用树脂纤维制作而成，可根据要求喷各色油漆。有男半身、女半身、小童半身。

# 插针包布男女全身

有成人男装和女装全身。由环保PU料一次性成型制成，可插针，包布，布色可选择，头可根据需要更换成概念头、铝头（饼）或木头。手可拆开，并可根据需要随意摆弄姿势。

# 男女全软模特

本产品由环保PU料经特殊处理加工制成，仿人设计各个关节，摸着有真人肌肉的感觉，柔软度好，韧性强，可以随意摆出各种动作，如坐、睡、骑车等等，也可以站立。还可根据需要选择各种包布颜色。

# 小童软体全身模特

环保PU料一次成型，有分手和连手两款。分手小童的手可装可拆；连手小童的手和身连成一体，像真人一样。两种小童的头都可根据需要随意搭配木头、铝头、概念头，身体可以像人一样随意摆弄各种姿势，非常便于服装的展示。

# 玻璃钢全身模特

用树脂纤维制作而成，可根据要求喷各色油漆。款式有成人男装、女装和小童，姿势有站、坐、躺、跪、趴等各种造型。手的肩关节可以动。

# 手套模特

手套模特由环保PU海绵发泡制成，腕部及手指都可以像真人手一股随意弯曲，可单独用于手套产品或手部饰品的展示。另外还有塑料手掌和木手掌。

# 裤子展示模特（裤模）

用高级环保PU料发泡制成，可包各种颜色布料，手感如真人，可像真人一样摆出各种动作，可站在底盘上，也可挂在橱窗或墙上，还可随意弯曲地坐在凳子上。玻璃钢和塑料制作的裤模以站模为主，不能改变姿势。

# 内衣模特

用于男女内裤和女士文胸的展示。展示内裤的以裤台为主，有PE材料和玻璃钢的，包括实用展示型和迷你型。玻璃钢的可以喷各种颜色，还可根据个人需要设计LOGO。展示文胸的多数用灯光模特：内置发光器，特别适用于内衣的夜间展示，在夜间展示比基尼和泳衣，特别醒目。

# 袜模

用来展示袜子。有玻璃钢和塑料两种材料的。 珠宝首饰展示道具：用环保PU发泡而成，或用玻璃钢制造，可根据需要包各种颜色的布。用来展示项链等各种珠宝首饰。

# 衬衫模

用于衬衫、领带、领结等服饰的展示，配头用来展示帽子或围巾也别有风味。
